Sistema Desktop

Sistema Hughes — EACE

Sistema de gestão de escolas para o programa EACE da Hughes Network Systems. Substitui um processo manual baseado em planilhas Excel por uma aplicação desktop conectada a um banco de dados em nuvem, acessível por toda a equipe em tempo real.

VB.NET Windows Forms Supabase REST API Python PostgreSQL
  • Consulta de escolas por código INEP com retorno em tempo real
  • Edição de responsáveis, telefones, provedor e coordenadas GPS
  • Integração direta com WhatsApp e Google Maps
  • Banco de dados em nuvem com Row Level Security (RLS)
  • Migração automatizada de dados via script Python
  • Instalador profissional distribuído via Inno Setup
